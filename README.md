# Sentiment Analysis on IMDB Reviews using BERT

## 📌 Project Overview
This project applies **BERT (Bidirectional Encoder Representations from Transformers)** to perform sentiment classification (positive/negative) on the [IMDB 50k Movie Reviews dataset](https://www.kaggle.com/datasets/lakshmi25npathi/imdb-dataset-of-50k-movie-reviews).

## 🧰 Tools & Libraries
- Python 3.x
- PyTorch
- Hugging Face Transformers
- Scikit-learn
- Google Colab (GPU runtime)
